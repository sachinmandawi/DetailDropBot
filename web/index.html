Cloudflare CORS Setup

To query directly from the browser securely without a proxy, add this response header to your Cloudflare Workers code:

return new Response(JSON.stringify(data), {
  headers: {
    "Content-Type": "application/json",
    "Access-Control-Allow-Origin": "*"
  }
});
